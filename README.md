
# Credit-Scoring-AI

##Project Title
Loan Approval Prediction using Machine Learning

##Description

This project builds a machine learning model to predict whether a loan application should be approved or not based on applicant details such as income, age, credit history, and loan amount.

## Objective

Understand credit scoring
Build a classification model
Predict loan approval
Analyze model performance
Dataset
The dataset contains:

Income
Age
Credit History
Loan Amount
Loan Status
Technology Used
Python
Pandas
Scikit-learn
Google Colab
Model Used
Logistic Regression

Workflow
Data Collection
Data Preprocessing
Train-Test Split
Model Training
Prediction
Evaluation
Results
The model predicts whether a loan is:

Approved (1)
Not Approved (0)
